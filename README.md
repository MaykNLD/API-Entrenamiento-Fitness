# API de Entrenamiento Fitness
Interfaz de programación REST para la gestión de rutinas de entrenamiento y ejercicios.

## Características
- Gestión completa de entrenamientos (CRUD).
- Arquitectura escalable y profesional.
- Persistencia de datos en formato JSON (preparado para integración con bases de datos).
- Documentación interna de rutas clara y semántica en español.

## Tecnologías Utilizadas
- **Node.js**
- **Express.js** (Framework web)
- **Body-parser** (Procesamiento de peticiones)
- **UUID** (Identificadores únicos)

## Estructura del Proyecto
- `/src/index.js`: Punto de entrada de la aplicación.
- `/src/controllers`: Lógica de control y respuesta.
- `/src/services`: Lógica de negocio y procesamiento de datos.
- `/src/database`: Modelos de datos y configuración de persistencia.
- `/src/v1/routes`: Definición de rutas de la API versión 1.
